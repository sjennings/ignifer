# Ignifer

OSINT MCP Server aggregating 7 data sources for Claude Desktop

## Overview

Ignifer is a Model Context Protocol (MCP) server that provides Claude Desktop with access to multiple OSINT (Open Source Intelligence) data sources through a unified interface.

## Requirements

- Python 3.10+
- uv (package manager)

## Installation

```bash
make install
```

## Development

```bash
# Run tests
make test

# Run linting
make lint

# Run type checking
make type-check

# Format code
make format

# Run all checks
make all

# Clean build artifacts
make clean
```
